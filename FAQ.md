# FAQ

## What is Memo-app?
Memo-app is a lightweight note-taking and memo management application for creating, organizing, and managing personal notes digitally.

---

## What features does Memo-app support?

Current features include:
- Create notes
- Edit notes
- Delete notes
- Search notes
- Persistent storage
- Timestamp tracking

---
